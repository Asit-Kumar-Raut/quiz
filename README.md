# Fun Learn Play — Kids Quiz Games

A colorful, animated learning app for children with multiple mini-games.

## Games

1. **Count & Pick** — Count emojis, choose the correct number picture (4 options).
2. **Number Words** — Count emojis, pick the matching word (ONE, TWO, …).
3. **Picture Spelling** — See a picture, pick the correct spelling (4 options).
4. **Color Fun** — Guess the color of each object.
5. **More or Less** — Which group has more items?

## Features

- Main home screen with a box for each game
- Medals after every round (gold / silver / bronze / star)
- **Play Again** shuffles new questions every time
- Progress bar at the top shows medals from each game played
- Confetti and animations when you finish

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
npx serve .
```

## Files

- `index.html` — App shell
- `index.css` — Styles and animations
- `js/data.js` — Questions and game config
- `js/app.js` — UI and game flow
- `*.jpg` — Number images for counting games
